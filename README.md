# Enrollment Project using MongoDB
The Enrollment streamlines student enrollment in academic components by managing departments, courses, sections, students, majors, and enrollment. 
This system ensures efficient handling of academic records, simplifies course selection, and facilitates accurate tracking of student progress through their chosen major using MongoDB database.

## Features
* Able to add, delete, update and track departments, courses, course sections, students and majors.
* Enroll students to courses and sections.

## Tech used
* **Database**: MongoDB
